# Interpete de Racket en Clojure


##
```sh
user => (load-file "racket.clj")
user => (repl)
(enter! "demo.rkt")
(enter! "jarras.rkt")
(breadth-first bc)
```

## Usage

 To run interpreter:
```sh
lein run
(enter! "demo.rkt")
(enter! "jarras.rkt")
(breadth-first bc)
```

 To run tests: 
 ```sh
 lein test
 ```
